# ⚔️ Guardians of the Ancient Kingdom

## Descripción
Este proyecto es un juego de combate por consola desarrollado en Python utilizando Programación Orientada a Objetos (POO).

El jugador puede elegir entre diferentes clases de personajes y enfrentarse contra enemigos controlados por la computadora.

## Objetivo del Proyecto
El objetivo es practicar conceptos fundamentales de Python como:

- Clases y objetos
- Herencia
- Encapsulamiento
- Polimorfismo
- Uso de funciones
- Manejo de lógica de juego

## Personajes Disponibles

### 🛡️ Guerrero
- Alta vida
- Ataque fuerte
- Habilidad especial: Golpe fuerte

### 🔮 Mago
- Usa maná para lanzar hechizos
- Daño mágico elevado

### 🏹 Arquero
- Ataques críticos aleatorios
- Mayor probabilidad de daño alto

## Mecánica del Juego
1. El jugador ingresa su nombre.
2. Selecciona una clase.
3. Se genera un enemigo aleatorio.
4. Ambos luchan hasta que uno pierde toda la vida.

## Tecnologías Utilizadas
- Python 3
- Librería `random`

## Estructura del Código
- Clase base `Personaje`
- Clases derivadas:
  - `Guerrero`
  - `Mago`
  - `Arquero`
- Función `batalla()`
- Función principal `main()`
